# Doc-Knowledge 产品需求文档

**创建时间**：2026-05-17
**版本**：v0.1.0

---

## 1. 产品概述

Doc-Knowledge 是一个文档知识提取工具，将 Office 文档（PDF/PPT/DOCX/XLSX）批量转换为结构化 Markdown 知识库，支持去重、价值提取，并导出到 Obsidian 或 MemoMind。

## 2. 目标用户

- 知识工作者：需要将历史文档整理为知识库
- 研究人员：需要从大量文档中提取有价值的内容
- 团队：需要共享文档并去重

## 3. 核心功能

### 3.1 文档转换（A→B）
- 支持格式：PDF、PPTX、DOCX、XLSX
- 保持目录结构
- 每个 Markdown 文件包含反向链接指向源文件

### 3.2 知识提取（B→C）
- 语义去重（TF-IDF + 余弦相似度）
- 版本合并（保留最新/最完整版本）
- 价值评分（长度、结构、新鲜度、关键词密度、独特性）
- 自动分类和标签

### 3.3 导出（C→目标）
- 导出到 Obsidian Vault
- 导出到 MemoMind API
- 标准 Markdown 输出

## 4. 非功能需求

- **性能**：1000 个文件在 10 分钟内完成转换
- **准确性**：PDF 文本提取准确率 > 95%
- **可用性**：CLI 命令简单直观，支持一键管道

## 5. 技术约束

- Python 3.10+
- Windows 兼容
- 可选依赖（按需安装重型包）
