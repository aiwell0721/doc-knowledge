## 📁 产品文档 vs 项目文档

| 类型 | 目录 | 受众 | 内容 |
|------|------|------|------|
| **产品文档** | `../docs/` | 用户、部署运维 | 安装指南、使用说明、CLI参考、故障排查 |
| **项目文档** | `./` (docs-project/) | 开发团队、产品经理 | 产品概念、业务逻辑、架构设计、详细设计、开发计划、测试文档 |
