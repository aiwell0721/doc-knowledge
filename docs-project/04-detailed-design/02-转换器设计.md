## 1. 基础接口

```python
class BaseConverter(ABC):
    """文档转换器基类"""
    
    @property
    @abstractmethod
    def supported_extensions(self) -> list[str]:
        """支持的文件扩展名列表"""
        pass
    
    @abstractmethod
    def convert(self, filepath: Path) -> str:
        """
        将文件转换为 Markdown 格式
        
        Args:
            filepath: 源文件路径
            
        Returns:
            Markdown 内容字符串
        """
        pass
```

## 7. 图片处理策略

### 文档内嵌图片

提取为独立文件 + Markdown 引用，存入 `{filename}/_images/` 子目录。

### 独立图片文件

直接复制，不生成 .md 包装文件。

## 8. 不支持的格式处理

原文件复制 + 轻量元数据包装（仅文本+链接，~300 bytes）。

### 文件类型分类

| 文件类型 | 示例 | 处理策略 |
|---------|------|---------|
| 支持转换 | `.pdf`, `.docx` | 转为 `.md`（Phase 1） |
| 图片 | `.png`, `.jpg`, `.gif` | 直接复制 |
| 其他 | `.mp4`, `.zip`, `.exe` | 复制 + 元数据包装 |
