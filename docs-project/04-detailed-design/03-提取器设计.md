## 1. 语义去重器

```python
class DocumentDeduplicator:
    """
    基于语义相似度的文档去重器
    
    算法: TF-IDF + 余弦相似度
    阈值: 默认 0.85（可配置）
    """
    
    def __init__(self, threshold: float = 0.85):
        self.threshold = threshold
        self.tokenizer = ChineseTokenizer()  # 复用 MemoMind tokenizer
    
    def deduplicate(self, documents: list[Document]) -> DeduplicationResult:
        """
        对文档集合进行去重
        
        Returns:
            DeduplicationResult:
            - kept: 保留的文档列表
            - deprecated: 被去重的文档列表（含相似文档映射）
        """
        # 1. 计算每个文档的 TF-IDF 向量
        vectors = self._compute_vectors(documents)
        
        # 2. 计算相似度矩阵
        similarity_matrix = self._compute_similarity(vectors)
        
        # 3. 聚类相似文档
        clusters = self._cluster(similarity_matrix, self.threshold)
        
        # 4. 每个聚类选择最佳文档
        kept = []
        deprecated = []
        for cluster in clusters:
            if len(cluster) == 1:
                kept.append(cluster[0])
            else:
                best = self._select_best(cluster)
                kept.append(best)
                deprecated.extend([d for d in cluster if d != best])
        
        return DeduplicationResult(kept=kept, deprecated=deprecated)
    
    def _select_best(self, cluster: list[Document]) -> Document:
        """
        从相似文档组中选择最佳版本
        
        优先级:
        1. 修改时间最新
        2. 内容最长（信息量最大）
        3. 标题质量最高
        """
        return max(cluster, key=lambda d: (
            d.modified_at,
            len(d.content),
            self._title_quality(d.title)
        ))
```

## 2. 价值评分器

```python
class ValueScorer:
    """
    文档价值评分器
    
    评分维度（各占 20%）:
    1. 长度权重：500字以上得分更高
    2. 结构权重：有标题层级 > 纯文本
    3. 新鲜度：文件修改时间越近得分越高
    4. 关键词密度：专业术语密度
    5. 独特性：与其他文档的差异化程度
    """
    
    def score(self, document: Document, corpus: list[Document] = None) -> int:
        """
        计算文档价值评分（0-100）
        
        Args:
            document: 待评分文档
            corpus: 文档集合（用于计算独特性）
            
        Returns:
            评分 0-100
        """
        length_score = self._length_score(document)
        structure_score = self._structure_score(document)
        freshness_score = self._freshness_score(document)
        keyword_score = self._keyword_score(document)
        uniqueness_score = self._uniqueness_score(document, corpus)
        
        return int(
            length_score * 0.2 +
            structure_score * 0.2 +
            freshness_score * 0.2 +
            keyword_score * 0.2 +
            uniqueness_score * 0.2
        )
```

## 4. 主提取流程

```python
class Extractor:
    """
    知识提取主流程
    
    B→C: 读取目录 B → 去重 → 评分 → 分类 → 输出到目录 C
    """
    
    def extract(
        self,
        input_dir: Path,
        output_dir: Path,
        threshold: float = 0.85,
        min_score: int = 30,
        keep_deprecated: bool = False
    ) -> ExtractionResult:
        """
        执行知识提取
        
        Args:
            input_dir: 输入目录（B）
            output_dir: 输出目录（C）
            threshold: 去重阈值
            min_score: 最低价值评分
            keep_deprecated: 是否保留旧版本
            
        Returns:
            ExtractionResult: 提取统计信息
        """
        # 1. 读取所有 Markdown 文件
        documents = self._load_documents(input_dir)
        
        # 2. 语义去重
        dedup_result = self.deduplicator.deduplicate(documents)
        
        # 3. 价值评分
        for doc in dedup_result.kept:
            doc.score = self.scorer.score(doc, dedup_result.kept)
        
        # 4. 过滤低价值文档
        kept_docs = [d for d in dedup_result.kept if d.score >= min_score]
        
        # 5. 自动标签
        for doc in kept_docs:
            doc.tags = self.tagger.tag(doc)
        
        # 6. 输出到目录 C
        result = self._write_documents(kept_docs, output_dir)
        
        if keep_deprecated:
            deprecated_dir = output_dir / "deprecated"
            self._write_documents(dedup_result.deprecated, deprecated_dir)
        
        return result
```

## 5. 图片引用修正

提取流程在写入 C 前调用 `_fixup_image_refs()`，将图片引用从 B 内的同级相对路径改为指向 B 的外部路径：

- **输入**：`](file_images/img1.png)` — 同级路径，在 B 内有效
- **输出**：`](../mirror/subdir/file_images/img1.png)` — 从 C 指回 B

深度计算规则：从 `.md` 所在位置穿越到 `mirror/` 所需的 `..` 层数 = 目录深度 + 1（`mirror/` 和 `extracted/` 是兄弟目录）。
