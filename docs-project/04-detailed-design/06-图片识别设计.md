## 1. 概述

在文档转换过程中，提取 PPTX/PDF 中的图片并使用大模型（VLM）进行文字识别。

---

## 2. 架构设计

```
转换流程
  ↓
提取图片 → 图片识别（可选）→ 生成图片描述
  ↓
合并到 Markdown
```

| 方案 | 类 | 依赖 | 成本 |
|------|-----|------|------|
| 大模型 API | `LLMVisionService` | OpenAI 兼容 API | 按调用计费 |

## 3. 配置设计

### 3.1 配置文件（`~/.doc-knowledge/config.yaml`）

```yaml
vision:
  enabled: false              # 默认关闭
  backend: "llm"              # llm | ocr | hybrid
  llm:
    api_url: "https://dashscope.aliyuncs.com/compatible-mode/v1"
    api_key: "${DASHSCOPE_API_KEY}"
    model: "qwen-vl-plus"
    timeout: 30
  ocr:
    lang: "ch"                # ch | en
    use_gpu: false
```

### 3.2 CLI 选项

```bash
# 启用图片识别
doc-knowledge convert <dir> --vision

# 指定配置文件
doc-knowledge convert <dir> --vision --config ~/.doc-knowledge/config.yaml

# 指定模型（覆盖配置）
doc-knowledge convert <dir> --vision --api-url <url> --api-key <key> --model <model>
```

## 4. API 兼容性

采用 OpenAI 兼容格式，支持多种提供商：

| 提供商 | API URL | 模型示例 |
|--------|---------|---------|
| 通义千问 | dashscope.aliyuncs.com | qwen-vl-plus, qwen-vl-max |
| OpenAI | api.openai.com | gpt-4o, gpt-4o-mini |
| Claude | api.anthropic.com | claude-sonnet-4 |
| DeepSeek | api.deepseek.com | deepseek-vl2 |
| 本地部署 | localhost:8000 | qwen2.5-vl, llama3.2-vision |

### 4.1 请求格式（OpenAI 兼容）

```python
{
    "model": "qwen-vl-plus",
    "messages": [{
        "role": "user",
        "content": [
            {"type": "text", "text": "请识别图片中的所有文字..."},
            {"type": "image_url", "image_url": {"url": "data:image/jpeg;base64,..."}}
        ]
    }],
    "max_tokens": 2000
}
```

## 6. 费用估算

| 场景 | 图片数 | 模型 | 单次成本 | 总成本 |
|------|--------|------|---------|--------|
| 10页 PPT | ~30 张 | qwen-vl-plus | ¥0.008/张 | ¥0.24 |
| 50页 PPT | ~150 张 | qwen-vl-plus | ¥0.008/张 | ¥1.20 |
| 100页 PDF | ~300 张 | qwen-vl-max | ¥0.03/张 | ¥9.00 |

**qwen-vl-plus 价格**：¥0.008/千 Token（输入），约 ¥0.008/张
**qwen-vl-max 价格**：¥0.03/千 Token（输入），约 ¥0.03/张
