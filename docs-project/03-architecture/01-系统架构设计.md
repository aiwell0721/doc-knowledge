# Doc-Knowledge 系统架构设计

**创建时间**：2026-05-17
**版本**：v0.1.0

---

## 1. 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                    Doc-Knowledge CLI                     │
├───────────┬──────────────┬───────────┬──────────────────┤
│ convert   │ extract      │ export    │ pipeline         │
│ (A→B)    │ (B→C)       │ (C→目标) │ (A→B→C→导出)    │
├───────────┼──────────────┼───────────┼──────────────────┤
│ • PDF→MD │ • 语义去重   │ •Obsidian │ • 一键完成       │
│ • PPT→MD │ • 版本合并   │ •MemoMind │ • 进度显示       │
│ • DOCX→MD│ • 价值评分   │ • Markdown│ • 错误恢复       │
│ • XLSX→MD│ • 自动标签   │           │                  │
└───────────┴──────────────┴───────────┴──────────────────┘
```

## 2. 模块设计

### 2.1 converters/（转换器）
| 模块 | 输入 | 输出 | 依赖 |
|------|------|------|------|
| PdfConverter | PDF | Markdown | PyMuPDF |
| PptxConverter | PPTX | Markdown | python-pptx |
| DocxConverter | DOCX | Markdown | python-docx |
| XlsxConverter | XLSX | Markdown | openpyxl |

### 2.2 extractors/（提取器）
| 模块 | 功能 | 算法 |
|------|------|------|
| Deduplicator | 语义去重 | TF-IDF + 余弦相似度 |
| Merger | 版本合并 | 时间戳 + 内容长度 |
| ValueScorer | 价值评分 | 启发式多因子评分 |
| Tagger | 自动标签 | 关键词提取 + 同义词映射 |

### 2.3 exporters/（导出器）
| 模块 | 目标 | 格式 |
|------|------|------|
| ObsidianExporter | Obsidian Vault | Markdown + frontmatter |
| MemoMindExporter | MemoMind API | Markdown → API |
| MarkdownExporter | 文件系统 | 纯 Markdown |

## 3. 数据流

```
目录 A (源文件)
    ↓ convert()
目录 B (MD镜像 + 反向链接)
    ↓ extract()
目录 C (去重后的知识文档)
    ↓ export()
目标 (Obsidian / MemoMind / Markdown)
```

## 4. 技术选型

| 组件 | 选型 | 理由 |
|------|------|------|
| CLI 框架 | click | 优雅、易扩展 |
| 进度显示 | rich | 美观、功能丰富 |
| PDF 解析 | PyMuPDF | 速度快、API 好 |
| PPT 解析 | python-pptx | 纯 Python |
| DOCX 解析 | python-docx | 成熟稳定 |
| Excel 解析 | openpyxl | 纯 Python |
| 去重/相似度 | 自建 TF-IDF | 零外部依赖 |
